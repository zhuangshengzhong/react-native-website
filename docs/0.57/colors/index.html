颜色

本文档贡献者：sunnylqm(95.63%), kt.tian(4.37%)

Components in React Native are styled using JavaScript. Color properties usually match how CSS works on the web.

Red-green-blue

红-绿-蓝

React Native 支持 rgb() 和 rgba() 两种十六进制与函数方法

  • '#f0f' (#rgb)

  • '#ff00ff' (#rrggbb)

  • 'rgb(255, 0, 255)'

  • 'rgba(255, 255, 255, 1.0)'

  • '#f0ff' (#rgba)

  • '#ff00ff00' (#rrggbbaa)

色调-饱和度-亮度

也支持 hsl() 和 hsla() 函数方法：

  • 'hsl(360, 100%, 100%)'
  • 'hsla(360, 100%, 100%, 1.0)'

透明度

rgba(0,0,0,0) 的快捷方式是：

  • 'transparent'
