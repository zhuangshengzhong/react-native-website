React Native 版本规则

开源版本的React Native原则上每月发布一个新版本。在每月初发布新的稳定版本的同时， 在GitHub的最新主代码分支上会切出一个新的测试候选版本（RC）。这个候选版本会在这一个月 中接受大家的尝鲜测试，并积极听取描述清楚的、有建设性的意见反馈。下个月时，这一候选版本就会成为新的稳定版本。

最新候选版本

要了解尝试最新的变化并提供积极的意见反馈，那就来试试最新的候选版本吧。 候选版本中的新变化一般会在两周之后正式版发布时，在Facebook的官方应用中上线使用。
